# Facebook

## Description
Welcome to The Facebook Project. It aims to provide the following functionalities:
1. Sign-up
    * Create Profile
2. Sign-in
    * Edit profile
    * Delete profile
    * View profile
    * View all profiles
    * Search profiles
    * Sort profiles by names
    * Create timeline

## Software required
> Java 1.8
> 
> JDBC
> 
> Eclipse
> 
> Oracle SQL
